# ⚙️ Technical Stack

## Frontend
- React (Next.js App Router)

## Styling
- Tailwind CSS
- Optional: shadcn/ui components

## Backend
- Python (FastAPI)

## AI / ML
- Microsoft Presidio (PII detection)
- spaCy (NER fallback)
- Regex-based detection

## Database
- PostgreSQL (main data)
- Redis (token vault + caching)

## Authentication
- NextAuth.js (SSO support)
- Google OAuth (MVP)

## Deployment
- localhost deployment

## APIs
- Gemini api 3.1 pro
